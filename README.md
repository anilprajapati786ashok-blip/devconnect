# 🚀 DevConnect 

DevConnect  is an AI-powered developer platform that helps users build portfolios, improve resumes, match jobs, and prepare for interviews — all in one place.

It is designed to help developers get internships and job opportunities faster using smart AI tools.

# 🤖 AI Career Suite

DevConnect  provides a complete AI-powered career preparation system to help developers get job-ready faster.

---

## 📄 Resume Analyzer & Improvement

Upload your resume and get smart AI feedback to improve it.

### 🔍 Features:
- Resume parsing (skills, experience, projects)
- ATS Score (0–100)
- Missing skills detection
- AI suggestions to improve resume
- Resume optimization tips

---

## 💼 Job Match System

Smart job recommendation engine based on your profile.

### 🔍 Features:
- Skill-based job matching
- Experience-based filtering
- Job compatibility score (% match)
- Internship & job recommendations

---

## 🎤 Mock Interview System

Practice real interview scenarios with AI-based feedback.

### 🔍 Features:
- Technical + HR interview questions
- AI-generated questions
- Answer evaluation system
- Feedback & improvement tips
- Retry practice mode
